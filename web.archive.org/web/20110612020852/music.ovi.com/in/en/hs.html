Welcome to Symphonia - Help & Setup

Getting Started

  • To register, go to the top-right corner and click Sign In. Then click Register.
  • Once logged in, your favorites, downloads, and recently played tracks will be saved.
  • Click on any song to start listening instantly.

Windows XP

  • Use the Supermium browser for best compatibility.
  • Enable Notifications in Supermium to receive song change alerts.
  • HTML5 Audio is supported in Supermium, but not in Firefox. Please avoid using Firefox on XP.

Windows Vista

  • Recommended Browser: Supermium or Chromium 87+ based forks.
  • Ensure JavaScript is enabled and page notifications are allowed.
  • Use the mobile version if you face layout issues.

Windows 7

  • Recommended Browser: Chrome, Supermium, or Opera 70+.
  • MS Media Session API is partially supported. If music details don’t show, enable notifications to get updates.
  • Use the desktop site for the best visual experience.

Browser Compatibility

  • Edge: Song details may not appear. Please enable notifications.
  • Firefox: Media Session API is not supported. Use Chrome-based browsers instead.
  • Supermium: Best for legacy Windows systems (XP, Vista).

FAQs

  • Why can't I download music?
    Symphonia streams legally available content. Downloads are not allowed on desktop to prevent piracy.
  • Why don't I see song name/artist info?
    This depends on your browser’s support for the Media Session API. Enable notifications to get info as popup alerts instead.
