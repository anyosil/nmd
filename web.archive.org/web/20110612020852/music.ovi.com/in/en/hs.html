Welcome to Symphonia - Help & Setup

Getting Started

  • To register, click Sign In on the top right, then choose Register.
  • Once signed in, your favorites, downloads, and recent plays are saved.
  • Click any song to start listening.

Windows XP

  • Use the Supermium browser for best support.
  • Enable Notifications in Supermium to get music info updates.
  • Avoid Firefox – it does not support MediaSession API.

Windows Vista

  • Supermium is still your best bet here.
  • Make sure notifications are turned on.
  • If layout issues occur, use the mobile site version.

Windows 7

  • Use Chrome or Supermium for best performance.
  • If song details don't appear, use browser notifications instead.

Browser Compatibility

  • Edge: Doesn't show song info well. Use notifications instead.
  • Firefox: Media Session API unsupported. Use Chrome-based browsers.
  • Supermium: Ideal for legacy systems like XP, Vista, 7.

FAQs

  • Why can't I download music?
    Symphonia only streams music to avoid piracy. Downloads are available only on specific mobile platforms.
  • Why doesn’t song info appear?
    Check if your browser supports Media Session API. If not, enable notifications to get updates.
