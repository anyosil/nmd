Welcome to Symphonia Help & Setup

Getting Started

  • To register, click on the Sign In button and select Create Account.
  • Your account details are saved securely on-device. You don't need to remember anything!
  • Check the "Always Logged In" option to skip login next time.

How to Listen to Music

  • Browse songs on the homepage or use the search bar to find your favorite tracks.
  • On desktop, songs are streamed directly from our servers — no downloads.
  • On supported mobile devices, tapping a song may start a direct download, only if allowed by the platform.
  • Symphonia doesn’t host pirated music — only royalty-free or user-submitted content is allowed.

Favorites, Downloads & Recently Played

  • Every time you listen to a song, it may be added to Recently Played.
  • Click the heart icon to add to Favorites.
  • My Downloads only appears on supported mobile browsers that allow downloading MP3s.
  • All data is stored using localStorage, so it stays with your browser!

Enabling Song Notifications

  • Supported browsers will show a small popup asking for permission — accept it!
  • Each time a song changes, a system notification will appear (if enabled).
  • Does not work on Firefox, Edge, or older mobile devices.

Browser & OS Setup

Windows XP
  • Use Supermium browser: supermium.com
  • Enable notifications when prompted.
  • Seekbar and music details won’t work on Firefox — use Supermium for full features.
Windows Vista
  • Use Supermium or Chrome 109 (last supported version).
  • Turn on system notifications in Supermium for best experience.
  • Older Edge and Firefox may not support media metadata — limited UI features.
Windows 7
  • Use a modern Chromium-based browser. Supermium or Chrome 109 is recommended.
  • Ensure notification permission is granted for full experience.
  • Some features may be unavailable in older browser versions.

FAQ

  • Why can't I download songs on PC?
    To comply with content regulations and copyright safety, downloads are restricted to supported mobile devices only.
  • Why don’t I see song details on Firefox or Edge?
    Some browsers don’t support the Media Session API. You’ll still hear music, but enhanced features are unavailable.
  • What’s the best browser for Symphonia?
    We recommend Chromium-based browsers like Chrome, Brave, or Supermium (especially on older systems).
